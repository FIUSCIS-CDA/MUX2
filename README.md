#### MODULE NAME
       MUX2

#### DEPENDENCIES
       None

#### INPUTS
Name | Size(Bits)
-----|------------
A   |     1      
B   |     1      
S   |     1      

#### OUTPUTS
Name | Size(Bits)
-----|------------
Y   |     1      

#### DESCRIPTION
       Two-input multiplexor with one-bit inputs.  Sets output Y equal to:

       A if S = 0,

       B if S = 1.
